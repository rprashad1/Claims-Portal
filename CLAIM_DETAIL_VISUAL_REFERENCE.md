## ?? Color Coding Reference

### Status Badges
```
[Open] - Green (bg-success)
[In Progress] - Blue (bg-info)
[Pending] - Orange (bg-warning)
[Closed] - Gray (bg-secondary)
```

### Summary Cards
```
Expense Reserve ? Info Blue
Indemnity Reserve ? Warning Orange
Total Reserves ? Primary Dark Blue
Count ? Success Green
```

### Claim Type Badges
```
[Driver] - Gray
[Passenger] - Gray
[ThirdParty] - Gray
```

## ?? Responsive Breakpoints

### Desktop (1920px+)
- All columns visible
- Horizontal scroll for wide tables
- Side-by-side cards
- Full feature set

### Tablet (768px-1024px)
- Compact columns
- Reduced padding
- Scrollable table
- Stacked cards

### Mobile (< 768px)
- Limited columns
- Full-width table
- Horizontal scroll
- Stack layout
- Touch-friendly buttons

### Grid 1: Sub-Claims Grid
Best for:
- Bulk selection
- Bulk actions
- Quick overview
- Managing multiple features at once

### Grid 2: Working Sub-Claims Grid
Best for:
- Detailed review
- Financial tracking
- Status monitoring
- Individual feature management
- Summary statistics

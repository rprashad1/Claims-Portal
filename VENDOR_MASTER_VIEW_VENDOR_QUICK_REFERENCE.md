## ?? What You Can View

### Basic Info
- Vendor name
- Type (Medical, Hospital, Attorney, etc.)
- FEIN number
- Effective date
- Status (Active/Disabled)

### Contact Info
- Contact person name
- Email (clickable)
- Business phone (clickable)
- Mobile phone (clickable)
- Fax number

### Addresses
- Main address
- Alternate addresses
- City, State, ZIP
- Address status

### Payment Schedule
- Frequency (Monthly/Weekly)
- Payment dates/days
- Display: "Monthly - Day 1, Day 15"

### Record Info
- Created date and who created
- Last updated date and who updated

## ?? Troubleshooting

### "Can't click fields"
? Normal! View is read-only. Click Edit to change.

### "Want to edit?"
? Close View modal, click Edit button.

### "Email/phone not clickable"
? Check if field has data. Links only work if filled.

### "Can't close modal"
? Click Close button or press Escape.

---

## ?? When to Use View

? **Use View When:**
- Just need to check information
- Want to reference details
- Verifying before editing
- Multiple people need to see same vendor
- Don't want to risk accidental changes

? **Don't Use View When:**
- You need to make changes
- You need to add/edit addresses
- You need to update payment schedule
- You're making bulk updates

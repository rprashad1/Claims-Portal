## Grid Column Details

### Full 14-Column Structure

```
Column 1:  ? (Checkbox - 40px)
Column 2:  FEATURES (250px+) - Feature # + Claimant Name
Column 3:  LIMIT ($) - Coverage Limit
Column 4:  DED ($) - Deductible
Column 5:  OFFSET ($) - Offset
Column 6:  ASSIGNED ADJUSTER (150px) - Adjuster Name
Column 7:  LOSS (Reserves) - Loss Reserve Amount
Column 8:  EXPENSE (Reserves) - Expense Reserve Amount
Column 9:  LOSS (Paid) - Loss Paid Amount
Column 10: EXPENSE (Paid) - Expense Paid Amount
Column 11: SALVAGE (Recovery) - Salvage Recovery Amount
Column 12: SUBROGATION (Recovery) - Subrogation Recovery Amount
Column 13: RECOVERED ($) - Total Recovered Amount
Column 14: STATUS - Status Badge (Open/Closed/Reopened)
```

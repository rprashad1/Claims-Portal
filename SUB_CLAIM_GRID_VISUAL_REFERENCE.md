## Pagination Controls

### Display Format

```
??????????????????????????????????????????????????????????????
? Records: 1 - 4 of 4  ?  Go to page: [1 of 1] [Export]    ?
??????????????????????????????????????????????????????????????
```

### Pagination Buttons

```
[First] - Go to first page
[<<]    - Previous 10 pages
[<]     - Previous page
[>]     - Next page
[>>]    - Next 10 pages
[Last]  - Go to last page
```

### Display Logic

```
If CurrentPage == 1:
  - First, <<, < buttons are disabled
  
If CurrentPage == TotalPages:
  - >, >>, Last buttons are disabled
  
Otherwise:
  - All buttons are enabled
```

## User Interactions

### Selecting a Row

```
1. User sees grid
   ?
2. Clicks checkbox on row
   ?
3. Row highlights
   ?
4. Checkbox shows ?
   ?
5. Action icons bar appears
```

### Selecting Multiple Rows

```
1. Click checkbox on row 1
   ? Action bar appears
2. Click checkbox on row 2
   ? (Action bar still visible)
3. Click checkbox on row 3
   ? (Action bar still visible)
4. Now have 3 rows selected
```

### Performing an Action

```
1. Multiple rows selected
   ?
2. Action bar visible
   ?
3. Click action icon (e.g., "Reassign")
   ?
4. Modal appears
   ?
5. Complete action
   ?
6. Action applied to all selected rows
```

### Deselecting

```
1. Click "Select All" header checkbox
   ? Deselects all rows
   OR
2. Click individual row checkbox
   ? Deselects that row
   ?
3. If all deselected
   ?
4. Action bar hides
```
